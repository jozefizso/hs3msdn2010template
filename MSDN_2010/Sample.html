Sample Document

Follow the simple steps below to add context sensitive popup help to a Visual Basic project.

  • Open the Visual Basic project you wish to develop context sensitive popup help for
  • Start Context! VB by selecting it from the Visual Basic Add-ins menu
  • Select a module from the list presented on the left of the screen
  • Select a control from the list presented in the grid at the bottom right of the screen and type in the required help text, either directly into the Grid or by using the large text box at the top right of the screen
    Note:

    If Control Tracking is enabled, you can simply select a control on the active form, and Context! VB will automatically select the control from the grid and display it's help text.
  • Repeat for all required controls and modules
  • Spell check the help text
  • When you are done, press the compile button on toolbar
  • Your compiled help file has been created in a sub-directory of your VB Project directory named 'ContextVB\Neutral'.  The file has automatically been copied into your project directory to enable you to test the help text.
  • Run your project and try out the popup help
    Note:

    Please note that Context! VB sets the WhatsThisHelp and WhatsThisButton properties to True for any forms containing controls for which you have entered help text when you compile the help.  This property setting is required to enable context sensitive popup help.
